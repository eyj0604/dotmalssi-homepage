# 기여 안내

도트말씨 홈페이지는 현재 상태와 승인된 운영 원칙을 정확하게 보여 주는 것이
우선입니다. 변경 제안은 한 가지 목적만 담은 작은 Issue와 Pull Request로
나눠 주세요.

## 변경 전 확인

- 프로젝트 상태 문구에는 기준일과 정본 근거를 적습니다.
- `완성`, `정식판`, `공식`, `검증 완료`를 증거 없이 사용하지 않습니다.
- 번역 제작자와 독립 검수자를 구분합니다.
- 승인된 `public/brand/` 자산을 재가공하거나 원작 표장과 결합하지 않습니다.
- ROM, BIOS, 세이브, 세이브스테이트, 비밀키, 개인정보를 포함하지 않습니다.

## 검사

```bash
npm ci
npm test
npm run lint
npm run typecheck
npm run security
```

공개 배포는 Pull Request 병합과 별개입니다. 정확한 배포 대상과 산출물에 대한
사용자 승인이 있어야 합니다.
