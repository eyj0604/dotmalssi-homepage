# 보안 정책

이 저장소는 정적 공개 홈페이지를 대상으로 하며 ROM·BIOS·세이브·자격 증명을
취급하지 않습니다.

보안 문제를 발견했다면 민감한 재현 정보나 비밀값을 공개 Issue에 올리지 말고
[저장소 소유자의 GitHub 프로필](https://github.com/eyj0604)을 통해 비공개 연락
경로를 먼저 요청해 주세요.

다음 항목은 즉시 차단 대상입니다.

- 자격 증명, 쿠키, 개인정보 노출
- 원본·패치 ROM 또는 세이브 파일 포함
- 승인되지 않은 다운로드나 외부 스크립트 삽입
- 링크를 통한 악성 파일 배포

브랜드, 번역문, 원작 자산의 권리 문의는 보안 취약점과 구분해 기록합니다.
