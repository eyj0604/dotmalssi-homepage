# 도트말씨 홈페이지 로드맵

이 문서는 확정된 기능 약속이 아니라, 사용자 승인과 권리·기술 검증을 거쳐
차례로 여는 공개 기능 계획이다. 한 작업에서는 한 프로젝트만 수정하고, 공개
사이트에는 ROM·세이브·비밀정보를 넣지 않는다.

## 1. 공개 정합화와 GitHub 정본

- 실제 공개 주소, 홈페이지 상태, 전용 GitHub 링크를 일치시킨다.
- 개인 이메일이 없는 `users.noreply.github.com` 커밋만 공개 이력으로 사용한다.
- 테스트·독립 검수·정확한 revision 승인 뒤에만 GitHub와 Sites를 갱신한다.

## 2. 게임 정보와 공략 라이브러리

게임별 공개 페이지는 다음 필드를 가진 작은 데이터 파일에서 생성한다.

- 제목, 원제, 플랫폼, 발매일, 개발·유통사, 장르
- 한글화 상태, 지원 원본 식별 정보, 다음 검증 게이트
- 작성자, 확인일, 출처 URL, 출처 유형, 이용허락·인용 근거
- 직접 작성한 소개와 공략 요약, 공식·허가된 상세 자료 링크

인터넷 자료를 검색해도 문장과 공략을 그대로 복사하지 않는다. 사실은 복수의
신뢰할 수 있는 출처로 확인하고, 설명과 공략은 도트말씨가 직접 작성한다. 외부
공략은 원문을 재게시하지 않고 합법성과 출처가 확인된 페이지로 연결한다.

## 3. 이미지와 영상

- 이미지: 직접 제작한 원본, 명시적 재사용 허가 또는 공개 라이선스가 있는 자산을
  우선한다. 직접 촬영한 검증 화면도 원작 요소를 담으면 자동으로 이용 권리가
  생기는 것이 아니므로, 허락·인용·공정이용 등 별도 근거와 최소 사용 범위를
  manifest에 기록한 경우에만 사용한다.
- 영상: 공식 채널이나 권리자가 허용한 플랫폼의 링크·공식 임베드를 우선한다.
- 게임 로고·캐릭터·박스아트·잡지 스캔·공략집 스캔을 권리 확인 없이 복사해
  저장소나 아트워크 팩에 넣지 않는다.
- 모든 미디어에는 게임 ID, 출처 URL, 확인일, 제작자, 라이선스, 허용 용도를
  가진 manifest를 둔다.

## 4. ROM과 패치 배포 경계

외부 링크라고 해서 저작권 위험이 사라지지는 않는다. 침해물임을 알면서 접근을
쉽게 하는 링크를 계속 제공하면 공중송신권 침해 방조가 문제될 수 있으므로,
불법 ROM 사이트·검색 결과·직접 다운로드 주소는 게시하지 않는다.

허용 후보:

- 제작사·유통사의 공식 판매 또는 공식 다운로드 페이지
- 권리자가 직접 무료 공개한 게임
- 재배포 라이선스와 권리자가 확인된 홈브루·공개 라이선스 게임
- ROM이 아닌 정확한 해시의 BPS·IPS·xdelta 패치와 적용 문서

금지:

- 권리 불명 ROM·ISO·BIOS·세이브·세이브스테이트 다운로드 링크
- 링크 단축·미러·검색어 안내로 불법 다운로드 위치를 우회 제공하는 행위
- 패치 ROM이나 원본 데이터가 포함된 아카이브

판단 근거는 [국가법령정보센터 저작권법](https://www.law.go.kr/LSW/lsInfoP.do?ancYnChk=0&chrClsCd=010202&efYd=20260511&lsiSeq=283335&urlMode=lsInfoP)과
[대법원 2021. 9. 9. 선고 2017도19025 전원합의체 판결 안내](https://www.scourt.go.kr/portal/news/NewsViewAction.work?gubun=4&pageIndex=1&searchOption=&searchWord=&seqnum=7935&type=0)를
기준으로 하되, 개별 게시물은 필요하면 별도 법률 검토를 받는다.

## 5. Attract-Mode 아트워크 팩

Attract-Mode의 표준 아트워크 이름인 `snap`, `marquee`, `flyer`, `wheel`,
`fanart`를 우선 지원한다. 공식 문서처럼 아트워크는 에뮬레이터별 경로에
연결하고 게임 선택에 따라 바뀌게 한다.

팩 제작 시 후보 구조:

```text
dotmalssi-attract-art-<system>-v<version>/
  snap/
  marquee/
  flyer/
  wheel/
  fanart/
  manifest.json
  CREDITS.md
  LICENSES/
  SHA256SUMS.txt
```

- 직접 만든 프레임·배경·한글 정보 패널을 중심으로 제작한다.
- 원작 로고·박스아트·스크린샷은 권리 또는 이용 근거가 확인된 경우만 넣는다.
- 파일명은 Attract-Mode romlist의 게임 식별자와 매칭하고, 실제 ROM은 포함하지 않는다.
- 정본 로고는 가독 크기와 여백 규칙을 지키며 원작 로고와 결합하지 않는다.

기술 기준: [Attract-Mode Layout and Plug-in Programming Reference](https://attractmode.org/docs/Layouts.html)

## 6. 빠르고 안전한 홈페이지 반영

목표는 “작업 후 바로 확인하고, 승인 후 빠르게 공개”이지 “모든 로컬 파일을
자동 업로드”하는 것이 아니다.

1. 수동 스프린트 세 번을 권한 위반과 상태 드리프트 없이 완료한다.
2. 읽기 전용 현황 생성과 미리보기부터 자동화한다.
3. PR에서 빌드·금지 파일·출처 manifest·링크 정책을 검사한다.
4. 공개 배포는 정확한 승인 revision만 1회 수행하고 kill switch를 유지한다.

자동화가 성숙해도 사람의 용어·표현·권리·배포 승인은 자동화하지 않는다.
